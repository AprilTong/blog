## 一元操作符 +

当+运算符作为一个一元操作符的时候，会调用 **ToNumber** 处理该值。

```js
console.log(+'1') // 1
console.log(+[]) // 0
console.log(+['1']) // 1
console.log(+['1', '2']) // NaN
console.log(+{}) // NaN
```
